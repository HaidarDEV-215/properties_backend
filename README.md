Property announcement program.
==============================

My program (which is A Fourth-Year University Semester Project at the Faculty of Information Engineering, University of Latakia.) aims to provide an easy, secure, and responsive user experience for properties listings. It includes an integrated account and user management system to ensure the application environment is as secure and reliable as possible.
Experience uploading secure and reliable data to the MongoDB database, managing it securely and reliably, and linking each property to its advertiser. With the ability to upload up to 10 property photos. And user profile pictures (Avatar) to display on their profile.
==============================

Tools I'd used:
Node js programming language,
MongoDB database,
postman to test my api's
==============================

To test my program, after download.
Just type:
>>npm install
in your directory console to install dependencies and required packages, and make sure you are connected to the internet.
After that, type:
>>npm run dev
Make sure to create <<.env>> file to store sensitive data, like(mongoDB_url, port and jwt secret code )
then it should work successfully
